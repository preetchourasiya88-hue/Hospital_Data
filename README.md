🏥 Hospital Dashboard Project – Power BI

I built a Hospital Management Dashboard in Power BI to analyze hospital operations and patient-related data through interactive reports and visualizations.
This dashboard helps in monitoring patient records, doctor performance, treatment statistics, revenue analysis, and overall hospital efficiency.
The main objective of this project was to convert raw healthcare data into meaningful insights that help hospitals improve decision-making and patient management.

📊 KPI Cards Used in the Dashboard
✅ Total Patients

Displays the total number of patients admitted or treated.

✅ Total Appointments

Shows the number of appointments booked.

✅ Total Revenue

Represents the overall revenue generated by the hospital.

✅ Average Treatment Cost

Shows the average cost per patient treatment.

✅ Bed Occupancy Rate

Tracks the percentage of occupied hospital beds.

✅ Doctor Count

Displays the number of doctors available.

✅ Patient Recovery Rate

Measures the percentage of recovered/discharged patients.

✅ Emergency Cases

Shows the total number of emergency admissions.

📈 Graphs & Visuals Used
📌 Bar Chart

Used to compare patient count, revenue, or department performance.

📌 Line Chart

Displays trends in patient visits, admissions, or revenue over time.

📌 Pie / Donut Chart

Shows department-wise patient distribution or treatment categories.

📌 Stacked Column Chart

Compares multiple hospital metrics like admissions vs discharges.

📌 Area Chart

Used to visualize patient growth trends over time.

📌 Table / Matrix Visual

Displays detailed patient records and department summaries.

📌 KPI Cards

Used for highlighting important healthcare metrics instantly.
