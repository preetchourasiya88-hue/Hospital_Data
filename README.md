🏥 Hospital Dashboard – Power BI Project Explanation

I created an interactive Hospital Dashboard in Power BI to analyze hospital operations, patient records, doctor performance, admissions, and discharge trends.
The main goal of this project was to convert hospital data into an easy-to-understand dashboard so that hospital management can quickly track performance and make better decisions.
Using Power BI (data modeling, DAX, KPI cards, charts, and slicers), I transformed raw hospital data into meaningful insights.

This dashboard helps monitor:

Patient admissions
Doctor performance
Bed availability
Appointment tracking
Discharge trends
Department performance

All important hospital insights are available in one dashboard.

📊 1. KPI (Key Performance Indicators) Used
👨‍⚕️ Total Patients

Shows the total number of patients in the hospital.

Why used:
Helps monitor overall patient volume.

🏥 Total Admissions

Displays how many patients were admitted.

Why used:
Tracks hospital admission activity.

✅ Total Discharges

Shows the number of discharged patients.

Why used:
Helps track recovery and patient flow.

🛏️ Bed Availability / Occupancy

Displays available and occupied beds.

Why used:
Helps manage hospital resources efficiently.

👨‍⚕️ Doctor Count / Doctor Performance

Shows number of doctors and patient handling by each doctor.

Why used:
Helps compare doctor workload and performance.

📅 Total Appointments

Shows appointments scheduled.

Why used:
Tracks hospital daily operations.

🏢 Department-wise Patients

Displays patients by department:

Cardiology
Orthopedics
Emergency
General Medicine etc.

Why used:
Identifies busiest departments.
